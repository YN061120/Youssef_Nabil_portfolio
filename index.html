Skills

I am passionate about creating projects that come to mind when I am alone. One time, I came up with the idea of developing an Encryption–Decryption Program that encrypts and decrypts any text entered by the user. I created it using C++, and it was a highly rewarding experience for me. I have also developed several other projects in the past.


The main idea behind the project’s encryption and decryption process relies on certain characteristics of text. Since gaining knowledge in the programming field, I have learned that every character and symbol — essentially anything that can be typed on a keyboard — is represented by what is known as ASCII code, which stands for American Standard Code for Information Interchange. From its name, we can conclude that it is meant to be a standard numerical representation of characters and symbols, allowing computers to process them efficiently.


My program uses this property by converting the numerical representation of characters into hexadecimal values for encryption, and as expected, the decryption process converts these hexadecimal values back into the original text.
